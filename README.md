# Mis cuentas — presupuesto personal

App de presupuesto personal con React + Tailwind + Supabase.

## Desarrollo local

1. `npm install`
2. Copia `.env.example` a `.env` y rellena tus claves de Supabase.
3. `npm run dev`

## Base de datos

El archivo `sql/schema.sql` crea la tabla `transactions`. Pégalo en el
SQL Editor de tu proyecto de Supabase antes de usar la app.

## Despliegue

Ver la guía paso a paso proporcionada junto con este proyecto para publicar
gratis en Vercel o Netlify.
